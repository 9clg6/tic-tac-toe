# Objectives

L'objectif est de créer un TicTacToe pour le test technique avec Betclic Group.

## Priorities

1. Avoir un TicTacToe fonctionnel en utilisant des widgets de base
2. Tester au maximum le code (TU, Integration, UI, [Optionnel: Golden])
3. Avoir une architecture propre et cohérente
4. Documenter le code
5. Migrer les interfaces vers du canvas
6. Création d'un menu de jeu
7. Mode online (deux device avec code de partie)
8. Mode online avec matchmaking
9. Optimisation finales de performances
10. Version web 
11. Version desktop
